## Data Points

| Field Name | Description |
|------------|-------------|
| `url` | The URL leading to detailed population information for a specific country. |
| `flag_image` | URL of the country's flag image for visual identification. |
| `country` | The name of the country for which the data is provided. |
| `last_year_population` | Total population of the country in the previous year. |
| `country_area` | Total area of the country in square kilometers. |
| `country_land_area` | Land area of the country in square kilometers excluding water bodies. |
| `country_density (/kmÂ²)` | Population density per square kilometer in the country. |
| `population_world_percentage` | The percentage of the world population residing in the country. |
| `country_population_rank` | The rank of the country based on its total population. |
| `population_by_year` | Annual population statistics over recent years for the country. |
| `annual_population_growth` | The annual growth rate of the country's population, including numeric change and percentage. |
| `births_per_day` | Average number of births occurring per day in the country. |
| `deaths_per_day` | Average number of deaths occurring per day in the country. |
| `emigrations_per_day` | Average number of people emigrating per day from the country. |
| `net_change_per_day` | Net daily change in the population after accounting for births, deaths, and emigration. |
| `population_change` | Overall change in population over the year. |
| `capital` | The capital city of the country. |
| `continent` | The continent on which the country is located. |
| `regions` | List of regions or subregions within the country, if applicable. |
| `larget_cities` | A list of the largest cities in the country by population. |
| `abbreviation` | The official abbreviation or ISO code of the country. |
